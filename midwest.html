Midwest Light and Power

A modern website for a regional utility company, built to simplify customer interaction and improve transparency. The primary goal was to create a professional and trustworthy online presence while making information easily accessible for all users.

HTML5 Tailwind CSS JavaScript PHP
View Live Project

Project Information

  • Category Web Design & Development
  • Client Midwest Energy Solutions
  • Project date 11 March, 2023

This project involved a complete overhaul of the company's digital front door. We focused on a mobile-first responsive design, ensuring a seamless experience across all devices and implemented a clear navigation structure to help users find information like outage maps and billing portals quickly.

The main challenge was to present a large amount of technical and administrative information in a way that was easy to understand for a non-technical audience. Balancing a modern, clean aesthetic with the serious, trustworthy branding of a utility company was also a key consideration.

We implemented a component-based design system using Tailwind CSS, allowing for rapid prototyping and consistent styling. Interactive elements were built with vanilla JavaScript to keep the site lightweight and fast and a simple PHP backend was used for the contact form and service alerts.

Key Features

  • Responsive Design
  • Service Outage Map
  • Online Bill Pay Portal
  • News & Alerts System
  • Customer Support Form

Loosely designed in Figma and coded in Visual Studio Code by yours truly. Built with HTML5, CSS3 and JavaScript, deployed with GitHub.
